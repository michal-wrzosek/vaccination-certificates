# vaccination-certificates

This is a very simple Decentralized Ethereum Application (DAPP) to provide solution for COVID-19 Vaccination Certificates.

Stack:
- truffle
- react

To run locally (enable Ganache before):
```
yarn
truffle migrate
cd client && yarn start
```
